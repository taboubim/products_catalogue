-- I used postgreSQL as DataBase
-- Tomcat as web container

- Deployment : Deploy 2 war files :
  1/ Front_module.war  using /site as application context
  2/ Rest_module.war using /services as application context



To use the app :
1/ http://localhost:8080/site/
2/ Click on Reset button to populate the data
3/ Load Catalogue to load all products
4/ Load Members : to load all members
5/ select products to add to a Member
6/ Select a Member
7/ Click on add selected products for selected member
8/ Click on LoadShopping bag
